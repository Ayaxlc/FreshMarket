# 🛒 FreshMarket

FreshMarket es un proyecto de desarrollo web que recrea la experiencia de un supermercado online mediante una interfaz moderna, intuitiva y totalmente responsive.

El objetivo de este proyecto ha sido seguir mejorando mis habilidades en desarrollo frontend, poniendo especial atención en el diseño, la organización del código y la experiencia del usuario.
